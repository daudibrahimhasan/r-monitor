# Research Collaboration Engine

This folder contains the isolated research workflow.

## Run

- `python -m reddit_intent_engine.main --config reddit_intent_engine/research/config.yaml`

## Layout

- Config: `research/config.yaml`
- Database: `research/data_clean/research_leads.sqlite`
- Log: `research/data_clean/logs.txt`
- Exports: `research/data_clean/exports/`

## Outputs

- `research/data_clean/exports/research_leads.csv`
- `research/data_clean/exports/research_leads.md`
